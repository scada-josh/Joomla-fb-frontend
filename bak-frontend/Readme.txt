MyBiz, One Page Responsive Business & Portfolio Theme

-------------------------------------------------------------------

Theme designed by @labdhimedia

MyBiz is a one page clean, modern responsive multi-purpose theme which is suitable for business, portfolio or creative websites. It includes 5 sub-sections with rich styling elements and unique features, such as portfolio, services and much more…


Features
==============
	Valid HTML5
	One Page Template
	Responsive Design
	Animated navigation w/ Fixed Navbar
	Animated Content Slider
	CSS3 goodness
	Services Section
	Portfolio Section
	HTML contact form
	Full-Width Footer
	Tooltips & Modal Window
	Bootstrap 2.1.1
	jQuery 1.7.2


Browsers
==============

	Firefox 4+
	Latest Chrome
	Latest Safari
	Internet Explorer 8+
	Opera 11
